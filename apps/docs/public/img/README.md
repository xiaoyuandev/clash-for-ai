This directory is reserved for public screenshots and static image assets used by the Starlight docs site.

Recommended conventions:

1. Use lowercase English file names.
2. Use kebab-case, for example `providers-page-overview.png`.
3. Avoid spaces and non-ASCII characters in file names.
4. Prefer PNG for UI screenshots.
5. Reference images from docs with absolute paths such as `/img/providers-page-overview.png`.
